# waseda_covit19_docs_backend

## init
**Please docker installed**
```
$ make init
```

## run app 
```
$ make server-run
```

## stop app 
```
$ make server-run
```

## run web server
```
$ make run
```

## run db
```
$ make db-run
```

## execute sql
**db is running**
```
$ make create-table sqlname=<sql file name>.sql"
```
